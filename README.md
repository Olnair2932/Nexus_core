# 🚀 NEXUS CORE

## Sistema de Inteligência Musical Multifonte

O **NEXUS CORE** é uma plataforma Node.js de busca, reprodução e organização musical com memória persistente, conversação inteligente e múltiplas fontes de áudio.

---

## ✨ Recursos

- 🎵 Biblioteca local de músicas
- 🔎 Busca automática em múltiplas fontes
- 🧠 Memória persistente de músicas
- ⭐ Sistema de preferências musicais
- 🤖 Integração Gemini para interpretação de comandos
- 💬 Modo conversa inteligente
- ☁️ Playlist sincronizada com Firebase
- 🔐 Login Google
- ▶️ Player Web integrado
- 🌐 API REST
- 🚀 Deploy compatível com Render
