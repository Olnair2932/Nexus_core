# NEXUS CORE

NEXUS CORE é um servidor Node.js para busca, reprodução e organização de músicas utilizando múltiplas fontes. O projeto possui biblioteca local, integração com Internet Archive, podcasts, memória persistente e sincronização de playlists com Firebase.

## Recursos

- Biblioteca local de músicas.
- Busca automática em múltiplas fontes.
- Internet Archive.
- Podcasts.
- Memória persistente.
- Playlist sincronizada no Firebase.
- Login com Google.
- Interface Web integrada.
- API REST.
