# 🚀 NEXUS CORE

## Sistema de biblioteca musical inteligente

NEXUS CORE é um sistema de gerenciamento e reprodução de músicas com interface web, comandos por chat, biblioteca local e sincronização em nuvem.

## ✨ Recursos

- 🎵 Biblioteca musical dinâmica
- 🔎 Busca e download de músicas
- ▶️ Player integrado no navegador
- 🤖 Comandos inteligentes pelo chat
- ☁️ Playlist sincronizada com Firebase
- 🔐 Login Google
- 🚀 Deploy automático no Render
